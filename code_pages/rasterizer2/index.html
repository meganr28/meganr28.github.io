Rasterizer and Camera in C++

October 2021

Description: A 2D and 3D rasterizer with an interactive perspective camera.

Features:

  • 2D triangle rasterization with line-segment intersection testing
  • Axis-aligned bounding boxes for optimization
  • Barycentric interpolation
  • Z-buffering
  • Interactive camera
  • Perspective-correct interpolation
  • Texture mapping
  • Lambertian reflection model

Extras:

  • Iridescent and toon shading model implementation
