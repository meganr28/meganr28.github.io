Micro Maya Mesh Editor in C++

October 2021

Description: A mesh editor application in the style of Autodesk Maya.

Features:

  • Half-edge mesh data structure implementation
  • Interactive GUI for selecting vertices, faces, and half-edges in the mesh
  • Topology editing functions, including split edge and triangulation
  • Catmull-Clark subdivision for mesh smoothing
  • Face extrusion
  • OBJ file importing

In-Progress:

  • Bind skeleton to mesh
  • Linear blend skinning
  • JSON file importing
