Mini Maya with Processing and Arduino

May 2020

VIEW PROJECT VIDEO HIGHLIGHTS

University of Virginia ARCH 5424 Final Project

Description: I created a Processing and Arduino project incorporating concepts from the entire semester, including cinematography, computer vision, and microcontrollers.

Features and Responsiblities:

  • Built a “mini Maya” 3D modeling and rendering program with Processing
  • Supports basic primitives, scaling, rotating, translating, and camera movement
  • Includes a pre-loaded animation when “play” button is pressed
  • Renders out frames to separate “data” folder in the current directory
  • While rendering, displays status on external LCD monitor (Processing sends signals to Arduino program)

Improvements:

  • Refactor code and make classes for certain portions - such as an Animation or Camera class
  • Test and add more components on the Arduino end
