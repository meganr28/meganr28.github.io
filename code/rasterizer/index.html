3D Rasterizer in Python

October 2020

Task Description: For my Introduction to Computer Graphics class at the University of Virginia, I created a 3D rasterizer that implements the pipeline for rasterizing triangles.

Features:

  • Transformations (model/view, projection, rotation, translation, scaling, and lookat)
  • Per-pixel clipping - pixels with a z-value less than 0 or greater than 1 are not drawn
  • Depth buffer check
  • Implements DDA algorithm for drawing lines
  • Supports sun lights coming from a specified direction
  • Allows an arbitrary number of lights and sums their contributions
  • Uses Lambert’s Law for illumination

Improvements:

  • Add support for textures and different types of lighting (e.g. point lights)

Code Sample:

  • rasterizer.py
  • chair.txt

To run:

python rasterizer.py chair.txt
