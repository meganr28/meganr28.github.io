Monte Carlo Path Tracer in C++

November 2020

Utah Teapot (50 samples)

Description: For my final project in Introduction to Computer Graphics at the University of Virginia, I extended a ray tracer I wrote into a unidirectional path tracer. I translated my original Python implementation into C++ for improved performance.

Features:

  • Monte Carlo sampling for global illumination
  • Sampling of area lights (produces soft shadows)
  • Color bleeding
  • Gamma correction
  • Ray-sphere, triangle, plane, and rectangle intersections
  • Diffuse, reflective, and refractive materials

Improvements:

  • Implement bidirectionality for more accurate caustics
  • Bounding volume hierarchy for acceleration

Code Sample:

For easier viewing, I have put all classes and dependencies in one .cpp file below:

  • pathtracer.cpp

If you would like to compile and run this project, download this zip file. It contains example files and the open-source image library I used to create png files. I reduced the sample rate for faster rendering times.

To compile:

g++ pathtracer.cpp -lpng -lX11 -lpthread -o output

To run:

./output spheres.txt
